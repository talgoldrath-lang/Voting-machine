# Voting Machine — הצבעה חיה

אפליקציית הצבעה בזמן אמת: שואלים שאלה בעל פה (או על שקף), הקהל מצביע מהטלפון
"כן" / "לא" / "תלוי", ומסך ההקרנה מציג את התוצאות המצטברות בזמן אמת —
ללא כל זיהוי של מי הצביע מה.

## הרצה

```bash
npm install
npm start
```

השרת עולה על `http://localhost:3000`.

- **מסך ההקרנה (למנחה):** `http://localhost:3000/host`
- **מסך ההצבעה (לקהל, מהטלפון):** `http://localhost:3000/`

הקהל צריך להיות באותה רשת / לגשת לכתובת ה-IP או לדומיין הציבורי של השרת.

## איך זה עובד

- מנחה פותח את `/host`, מקליד (רשות) את טקסט השאלה ולוחץ **"התחל הצבעה
  חדשה"** — זה פותח סבב הצבעה חדש ומאפס את כל הקולות הקודמים.
- כל משתתף פותח את `/` בטלפון ובוחר כן / לא / תלוי. אפשר לשנות הצבעה כל עוד
  ההצבעה פתוחה.
- מניעת הצבעה כפולה: לכל מכשיר נוצר מזהה אקראי שנשמר ב-`localStorage`
  בדפדפן. הצבעה נוספת מאותו מכשיר מחליפה את ההצבעה הקודמת שלו ולא נספרת
  פעמיים.
- מסך ההקרנה מציג רק ספירות מצטברות (עמודות אחוזים) — אין שום מידע על זהות
  המצביעים.
- כפתורי "סגור הצבעה" / "פתח מחדש" / "אפס תוצאות" מאפשרים למנחה לשלוט בסבב.

הנתונים נשמרים בזיכרון בלבד (אין מסד נתונים) — סבב הצבעה מתאפס אם השרת
מופעל מחדש.

## פריסה (Deploy)

האפליקציה היא שרת Node.js רגיל (Express + Socket.IO) — אפשר לפרוס אותה
בכל שירות שתומך ב-Node, למשל Render, Railway או Fly.io. יש להריץ
`npm install` ואז `npm start`.
